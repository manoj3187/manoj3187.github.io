Animal Shelters in the Neighborhood

Centralized Animal Rescue hosts information about Animal Shelters in the user's neighborhood. Users can look up shelter information and view animals available for adoption and fostering.


1. Sort by Rating and Distance

Similar to animal clinics, users can sort shelter results based on distance and ratings.





2. Search by name

The search by name features allows users to type in a keyword and brings up results that match the keyword
