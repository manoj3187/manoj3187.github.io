FORECASTING CONFIRMED CASES AND FATALITIES OF COVID-19 (PYTHON/TABLEAU/API- JHU)

• Forecasted confirmed cases and fatalities of Covid-19 using travel pattern and Google Trends data provided by John Hopkin’s University.

• Used Linear Regression, Random Forest Regression and Decision Tree Regression algorithms to achieve a r2 score of 0.85 and RMSE of 2.36.

• Pipeline was created on Spark. Tableau, Pyspark, Spark SQL and Spark ML were used in various stages of the project.


PREDICTING IMDB RATINGS OF A TV SHOW (PYTHON/SQL/TABLEAU/WEB-SCRAPING/NLP)

• Used Naïve Bayes algorithm to classify Sponge Bob episodes as ‘good’, ‘bad’ and ‘average’ depending on the rating with an accuracy of 80%.

• Features for each episode were derived by scraping transcripts of episodes from Spongebob.fandom.com.

• Mined text was then converted into sentiment scores for each character and writer of the episode using NLP (sentiment analysis) ▫ Modelling and EDA was carried out using Spark and Tableau respectively.


SCHOOL DATABASE (SQL, ORACLE 19C)

• Built a prototype database of schools using SQL on Oracle 19c.

• Database was tested from an admin’s, student’s and parent’s point of view by inserting and manipulating sample data using SQL queries.

• Built pipeline for effective querying of data. Some of the areas covered were admissions, student and teacher information, and curriculum data.


ANALYZING THE GROWTH OF A SOCCER CLUB (SQL/TABLEAU/API - FIFA)

• Used Tableau to analyze data acquired from the virtual soccer game FIFA 19 to determine factors that affect the growth of a soccer club over-time.

• Some of the factors that had a major impact were club revenue, player salaries, average age, club product sales, player transfers and commercial promotions.


IN-GAME CHAT PROTOCOL (IGCP)

• Built a chat protocol for the FIFA community using sockets in Python.

• The IGCP protocol allows gamers to chat with each other while making sure all participants adhere to community guidelines.
